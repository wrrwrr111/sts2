变形灵林谷

Act 1 - Overgrowth · Event

卡牌 角色 药剂 遗物 事件 怪物
事件流程 ⌄
初始场景 ⌄
场景
你进入了一片结晶树的林谷，树木开始激动地颤抖起来！
一阵阵杂乱的问候与欢迎声在你耳边不断响起，原来是一群变形灵从树上绽放了出来。

有一只变形灵在角落里坐立不安的样子，显然不像其他同类那样善于社交。

你是去找一群变形灵，还是去找落单的那个？
1
选择
大群变形灵
即时效果
[Is Multiplayer]失去所有金币。变化2张牌。
→ 结果 GROUP
你向那群叽叽喳喳的变形灵问好，它们伸出水晶手臂拥抱住了你。

它们将自己共有的知识分享给了你，更为你注入了拟形魔法，改变了你的本质……这些变形灵还真是群可爱的小生物啊。

其中一个偷走了你的金币。
2
选择
落单变形灵
即时效果
获得5点最大生命。
→ 结果 LONER
你向那只孤单的变形灵伸出一只手，同情它内向的天性。
它送给了你一颗小小的水晶果实，你深知它们的传统，于是当场把果子吃了下去。

水晶果实让你的心智清明、使命感也更加坚定。你道了谢，再次向上方出发。
